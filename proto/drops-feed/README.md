# Option A — Drops Feed (WhatsApp-post style)

## Concept
A warm editorial feed that feels like curated WhatsApp drops converted into a premium archive. Each card reads like a post: collage, chips, included titles, scarcity cue, and reservation CTA.

## Why it fits SBD
- Mirrors current drop behavior (3–4 times/week)
- Bundle-first storytelling without e-commerce clutter
- Strong scarcity and conversational tone

## Notes
- Mobile-first card stacking
- Minimal UI chrome; avoids "mini-Amazon" grid feel
